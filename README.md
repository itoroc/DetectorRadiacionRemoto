La solución propuesta es un equipo tecnológico de detección y localización de fuentes de radiación ionizante de tipo electromagnética (gamma y rayos X), cuya característica principal es su capacidad de despliegue remoto, al estar conformado por 2 elementos de hardware comunicados inalámbricamente entre sí: La Consola de Control y visualización, que consiste en una computadora embebida montada en una maleta compacta, que incorpora una pantalla táctil y que da acceso a un dashboard interactivo con información en tiempo real de los parámetros recibidos desde el segundo elemento de hardware, el Detector Remoto, el cual consiste en un microcontrolador con un sensor de radiación ionizante y un módulo de GNSS, el cual se comunica mediante tecnología LoRa y MQTT con la consola de control y cuenta con un sistema de montaje para vehículos no tripulados, tipo riel picatinny. En cuanto a la interfaz gráfica, esta muestra un mapa de radiación, cuya intensidad se representa en diferentes colores sobre el mapa, otorgando la capacidad de conocer el estado radiológico de un entorno durante una intervención de emergencia radiológica, mostrando también valores de tasa de dosis y cuentas por minuto, entregando información útil para la toma de decisiones por parte de los primeros intervinientes.
